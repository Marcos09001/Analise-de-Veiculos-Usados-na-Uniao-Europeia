# Analise-de-Veiculos-Usados-na-Uniao-Europeia
Este projeto tem como objetivo realizar uma Análise Exploratória de Dados (EDA) sobre o mercado de veículos usados na União Europeia, utilizando técnicas de limpeza de dados, análise estatística e visualização de dados.

A análise busca identificar padrões de precificação, características dos veículos e fatores que influenciam o valor de mercado, utilizando um dataset público contendo informações sobre veículos vendidos em diferentes países europeus.

O projeto inclui etapas completas de pré-processamento de dados, análise exploratória e construção de um dashboard interativo no Power BI, permitindo explorar visualmente os principais insights obtidos.

🎯 Objetivos da Análise

A análise busca responder algumas perguntas importantes:

1. Como os preços dos veículos usados se distribuem no mercado europeu?
2. Quais características influenciam mais o preço de um veículo?
3. Existe relação entre quilometragem e valor do carro?
4. O número de proprietários impacta no preço?
5. Carros com histórico completo de manutenção possuem maior valor?
6. Existem diferenças de mercado entre os países analisados?
